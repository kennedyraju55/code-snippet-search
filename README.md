## What It Does

- Search codebases semantically using natural language queries with Gemma 3
- Find relevant code snippets instantly without fuzzy matching limitations
- All processing happens locally — your code never leaves your machine

## Tech Stack

Python, Ollama, Gemma 3, Vector Search

## Quick Start

1. Clone: git clone https://github.com/kennedyraju55/code-snippet-search.git && cd code-snippet-search
2. Install: pip install -r requirements.txt
3. Pull model: ollama pull gemma3:4b
4. Index and search: python search.py

## Architecture

Uses Gemma 3 embeddings via Ollama to create semantic search indices, enabling natural language queries across code repositories.

## Why Local?

- **No API Keys Required** — Zero external dependencies
- **Complete Privacy** — Code stays on your machine
- **Offline Operation** — Works without internet
- **Cost-Free** — No monthly API bills
